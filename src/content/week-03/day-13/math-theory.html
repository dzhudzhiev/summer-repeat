Многочлены: сложение и вычитание

Если одночлен — это одно «слово» алгебры, то многочлен — это целое «предложение». Многочлены встречаются при решении уравнений, в физических формулах, в экономических расчётах. Умение складывать и вычитать многочлены — база для всей алгебры.

Многочлен и его стандартный вид

Многочлен (или полином) — это сумма одночленов. Одночлены, входящие в состав многочлена, называются его членами.

Пример: $3x^2 + 2xy - 5y^2 + 1$ — многочлен из четырёх членов. Обратите внимание: вычитание можно рассматривать как сложение с отрицательным одночленом.

Многочлен записан в стандартном виде, если выполнены три условия:

  1. Каждый его член — одночлен стандартного вида
  2. Нет подобных членов (все приведены)
  3. Члены расположены в порядке убывания степеней

Пример стандартного вида: $3x^2 + 5x - 7$ (степени: 2, 1, 0 — убывают).

Пример нестандартного вида: $3x^2 - 7 + 5x$ (члены не упорядочены по убыванию степени).

Степень многочлена

Степень многочлена — наибольшая из степеней одночленов, входящих в него. Чтобы найти степень, нужно определить степень каждого члена и выбрать максимальную.
Задача: Определите степень многочлена $3x^3y^2 - 2xy + 5$.

Решение:

Шаг 1. Находим степень каждого члена:

  • $3x^3y^2$: степень $3 + 2 = 5$
  • $-2xy$: степень $1 + 1 = 2$
  • $5$: степень $0$ (константа)

Шаг 2. Выбираем наибольшую степень: 5.

Ответ: степень многочлена равна 5.

Приведение подобных членов

Чтобы привести подобные члены, складывают их коэффициенты, а буквенную часть оставляют без изменения.

$$3x^2 + 2x - x^2 + 5x = (3 - 1)x^2 + (2 + 5)x = 2x^2 + 7x$$
Задача: Приведите многочлен $4a^2b - 3ab + 2a^2b + 5ab - a^2b$ к стандартному виду.

Решение:

Группируем подобные: $4a^2b + 2a^2b - a^2b = 5a^2b$ (члены с $a^2b$).

$-3ab + 5ab = 2ab$ (члены с $ab$).

Ответ: $5a^2b + 2ab$.

Совет: Подчёркивайте подобные члены одинаковыми линиями: одной чертой — члены с $x^2$, двумя — с $x$, волнистой — свободные члены. Это поможет не запутаться при приведении большого количества слагаемых.

Сложение и вычитание многочленов

Чтобы сложить (или вычесть) многочлены, нужно:

  1. Раскрыть скобки (при вычитании — изменить знаки каждого члена второго многочлена на противоположные!)
  2. Привести подобные слагаемые
Внимание на знаки!

Самая частая ошибка — забыть поменять знаки при вычитании. Если перед скобкой стоит минус, то:

$$-(a - b + c) = -a + b - c$$

Каждый знак внутри скобки меняется на противоположный: плюс становится минусом, минус — плюсом.

Задача: Сложите многочлены $(3a + 2b - c) + (a - 3b + 2c)$.

Решение:

Шаг 1. Раскрываем скобки (знаки не меняем, так как это сложение): $3a + 2b - c + a - 3b + 2c$.

Шаг 2. Группируем подобные: $(3a + a) + (2b - 3b) + (-c + 2c)$.

Шаг 3. Вычисляем: $4a - b + c$.

Ответ: $4a - b + c$.

Задача: Вычтите многочлены $(5x^2 + 3x - 1) - (2x^2 - 4x + 3)$.

Решение:

Шаг 1. Раскрываем скобки, меняя знаки второго многочлена: $5x^2 + 3x - 1 - 2x^2 + 4x - 3$.

Шаг 2. Группируем подобные: $(5x^2 - 2x^2) + (3x + 4x) + (-1 - 3)$.

Шаг 3. Вычисляем: $3x^2 + 7x - 4$.

Ответ: $3x^2 + 7x - 4$.

Сложение и вычитание многочленов «столбиком»

Многочлены можно складывать столбиком — как обычные числа. Для этого записывают одинаковые степени друг под другом:

$$\begin{array}{r} 3x^2 + 2x - 5 \\ + \quad x^2 - 3x + 2 \\ \hline 4x^2 - x - 3 \end{array}$$

Этот способ особенно удобен при большом количестве членов, когда легко запутаться в подобных слагаемых.

Геометрия: углы

Параллельно с алгеброй вспомним основы геометрии — они пригодятся для решения задач.

Угол — геометрическая фигура, образованная двумя лучами, выходящими из одной точки (вершины угла). Измеряется в градусах ($^\circ$).

Виды углов:

  • Острый — меньше $90^\circ$
  • Прямой — равен $90^\circ$
  • Тупой — больше $90^\circ$ и меньше $180^\circ$
  • Развёрнутый — равен $180^\circ$

Смежные углы — два угла, у которых одна сторона общая, а две другие являются продолжением одна другой. Сумма смежных углов равна $180^\circ$.

$$\angle AOB + \angle BOC = 180^\circ$$

Вертикальные углы — углы, стороны одного из которых являются продолжением сторон другого. Вертикальные углы равны.

Знание свойств углов помогает решать многие геометрические задачи: находить неизвестный угол, доказывать параллельность прямых, определять вид треугольника.

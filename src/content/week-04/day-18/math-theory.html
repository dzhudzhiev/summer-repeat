Решение уравнений: сложные случаи

Линейные уравнения редко встречаются в "чистом" виде. Чаще уравнение содержит дроби, скобки, модули или буквенные коэффициенты (параметры). Умение сводить такие уравнения к простым линейным — важнейший навык для успешной сдачи экзаменов.

Уравнения с дробями

Когда уравнение содержит дроби, главная идея — избавиться от них, умножив обе части на общий знаменатель.

Алгоритм:

  1. Найти общий знаменатель всех дробей
  2. Умножить обе части уравнения на общий знаменатель
  3. Решить полученное линейное уравнение
Задача: Решите уравнение $\frac{x}{3} - \frac{x}{4} = 2$.

Решение:

Шаг 1. Находим общий знаменатель: для 3 и 4 это 12.

Шаг 2. Умножаем обе части на 12:

$12 \cdot \frac{x}{3} - 12 \cdot \frac{x}{4} = 12 \cdot 2$

Шаг 3. Сокращаем: $4x - 3x = 24$

Шаг 4. Приводим подобные: $x = 24$

Ответ: $x = 24$.

Уравнения с модулем

Модуль числа $|x|$ — это расстояние от точки $x$ на числовой прямой до нуля. Модуль всегда неотрицателен.

Уравнение $|x| = a$ при $a > 0$ имеет два корня: $x = a$ и $x = -a$. Если $a = 0$, то один корень $x = 0$. Если $a < 0$, то корней нет (модуль не может быть отрицательным).

Задача: Решите уравнение $|x| = 5$.

Решение:

$|x| = 5$ означает, что расстояние от $x$ до 0 равно 5.

На числовой прямой таких точек две: $x = 5$ и $x = -5$.

Ответ: $x = 5$ или $x = -5$.

Уравнения с параметром

Параметр — буквенный коэффициент, значение которого не задано, но может влиять на решение уравнения.

Для линейного уравнения $ax = b$:

УсловиеКорни
$a \neq 0$$x = b/a$ (один корень)
$a = 0$, $b = 0$$x$ — любое число (бесконечно много)
$a = 0$, $b \neq 0$корней нет
Задача: При каком $a$ уравнение $ax + 3 = 7$ имеет корень $x = 2$?

Решение:

Шаг 1. Подставляем $x = 2$ в уравнение: $a \cdot 2 + 3 = 7$.

Шаг 2. Решаем полученное уравнение относительно $a$: $2a + 3 = 7$, $2a = 4$, $a = 2$.

Ответ: $a = 2$.

Уравнения, сводящиеся к линейным

Некоторые уравнения не выглядят линейными, но после преобразований становятся таковыми. Ключевой приём — использовать свойство пропорции.

Свойство пропорции: Если $\frac{a}{b} = \frac{c}{d}$, то $a \cdot d = b \cdot c$. Это правило позволяет избавиться от знаменателей за один шаг.
Задача: Решите уравнение с пропорцией: $\frac{x + 2}{3} = \frac{2x - 1}{5}$.

Решение:

Шаг 1. Используем основное свойство пропорции: $5(x + 2) = 3(2x - 1)$.

Шаг 2. Раскрываем скобки: $5x + 10 = 6x - 3$.

Шаг 3. Переносим слагаемые: $5x - 6x = -3 - 10$.

Шаг 4. Приводим подобные: $-x = -13$, значит $x = 13$.

Проверка: $\frac{13 + 2}{3} = \frac{15}{3} = 5$ и $\frac{2 \cdot 13 - 1}{5} = \frac{25}{5} = 5$. Верно.

Ответ: $x = 13$.

Решение задач с помощью уравнений

Уравнения — мощный инструмент для решения текстовых задач. Главное — правильно перевести условие с русского на язык алгебры.

Алгоритм:

  1. Обозначить неизвестное (обычно через $x$)
  2. Выразить другие величины через $x$
  3. Составить уравнение по условию задачи
  4. Решить уравнение
  5. Проверить ответ по смыслу задачи (отрицательная длина? дробное количество людей? — значит, ошибка)
Совет: При составлении уравнения обращайте внимание на ключевые слова: "сумма" ($+$), "разность" ($-$), "произведение" ($\cdot$), "в ... раз больше" ($\cdot$), "на ... меньше" ($-$), "всего" ($=$).

Квадратичная функция. График параболы

Квадратичная функция — функция вида $y = ax^2 + bx + c$, где $a \neq 0$.

Свойства параболы

1. Ветви:

  • $a > 0$: ветви направлены вверх
  • $a < 0$: ветви направлены вниз

2. Вершина параболы:

$$x_0 = -\frac{b}{2a}, \quad y_0 = y(x_0) = \frac{4ac - b^2}{4a} = -\frac{D}{4a}$$

3. Ось симметрии: прямая $x = x_0$.

4. Нули функции: корни уравнения $ax^2 + bx + c = 0$.

5. Точка пересечения с $Oy$: $(0, c)$.

Построение графика

Алгоритм:

  1. Найдите координаты вершины $(x_0, y_0)$.
  2. Проведите ось симметрии $x = x_0$.
  3. Найдите нули функции (если $D \ge 0$).
  4. Найдите точку пересечения с $Oy$: $(0, c)$.
  5. Возьмите 2-3 дополнительные точки.
  6. Постройте параболу по точкам.
Пример: $y = x^2 - 4x + 3$

$x_0 = 2$, $y_0 = -1$. Вершина: $(2, -1)$.

Нули: $x_1 = 1$, $x_2 = 3$.

$Oy$: $(0, 3)$.

Сдвиги параболы $y = x^2$

ФункцияПреобразование
$y = x^2 + c$Сдвиг вверх/вниз на $|c|$
$y = (x - m)^2$Сдвиг вправо/влево на $|m|$
$y = ax^2$Сжатие/растяжение
$y = -x^2$Отражение относительно оси $x$
